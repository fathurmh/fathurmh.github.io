Website link: https://fathurmh.github.io/

### Installing
**Clone this repository.**
```
$ git clone https://github.com/fathurmh/fathurmh.github.io/
$ cd fathurmh.github.io
$ git checkout -b development
```

**Install node modules.**
```
$ npm install
```

**Run application.**
```
$ npm start
```

**Deploy application.**
```
$ npm run deploy
```
